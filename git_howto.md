# Подсказки по GIT

## Создание репозитория:
```sh
git init
```
## Сохранение текущего состояния проекта в истории коммитов.
```sh
git add
```
## Фиксация измений.
```sh
git commit -m
```

## Просмотр историю коммитов. 
```sh
git log 
```

##
```sh
git log --oneline
```
## Переключение между коммитами
```sh
git checkout
```
