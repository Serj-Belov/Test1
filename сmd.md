# Подсказки в командной строке.

Команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории. MacOS, Linux
```sh
pwd
```

Листинг текущей директории Windows:
```sh
dir
```

Linux, MacOS:
```sh
ls
```

Удаление файлав Windows:
```sh
del <filename>
```

в Linux, MacOS:
```sh
rm <filename>
```
